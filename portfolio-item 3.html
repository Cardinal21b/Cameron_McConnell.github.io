Coding Portfolio

Portfolio Programming with HTML, CSS, and Javascript

As a mechanical engineering student pursuing Industrial Engineering with an emphasis in control systems, it is important that I have experience in software developement, and am able to adapt to different languages and systems. This is my first large project using HTML, CSS, and Javascript.

Using a Youtube Video called "Build a Portfolio Website (citation below), I built this website from scratch on VSC and altered some of the script to work with my portfolio needs. This includes things like creating a mission page and changing the design of the introduction section and menu icon.

It is important not to reinvent the wheel, and part of my engineering mantra is modifying and upgrading systems instead of attempting to rebuild them from scratch. This is represented by my use of a tutorial to understand the structure and syntax of html, css and javascript while modifying and editing components once I have gained an understanding of the inner workings of the language.

Classes in C++ at Concordia engineering and personal education in python aided in my understanding of the project.
